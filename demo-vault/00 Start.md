Welcome to the [Insert Multiple Attachments](https://github.com/mnaoumov/obsidian-insert-multiple-attachments/) demo vault. Out of the box, Obsidian's **Insert attachment** picker only lets you add **one** file at a time. **Insert Multiple Attachments** adds a command that opens a multi-select file picker, so you can insert **several attachments at once** - each as its own embed - with optional prefix, delimiter, and suffix text around them.

**How to try it:** open [[Playground]], place your cursor where you want the attachments, run **Insert Multiple Attachments: Invoke** from the Command Palette, and select two or more files in the OS picker (you can use the bundled `_assets/sample-attachments/` files, or your own images). Obsidian inserts an embed for every file you chose.
